# Endpoint Registry

The `UpstoxEndpointRegistry` serves as a central catalog for all API endpoints.

## Features
- **Centralized Listing**: Aggregates endpoints from all 7 categories (Authentication, Orders, Market Data, etc.).
- **Statistics**: Provides counts of endpoints by category and HTTP method.
- **Lookup**: Allows programmatic lookup of endpoints by Path or Method.

## Usage
Useful for dynamic API clients or for validating endpoint paths during development.

### Statistics (as of v2.0.0)
- **Total Endpoints**: 60+
- **Categories**: 7
- **Primary Methods**: GET, POST, PUT, DELETE
