VEGA TRADER – TERMINAL UI PAGES & STYLING GUIDE

(Bloomberg-grade, professional, dense, deterministic)

GLOBAL TERMINAL DESIGN SYSTEM (APPLIES TO ALL PAGES)
Color & Typography

Theme: Dark only

Background: #0B0E11 / #10141A

Primary Text: #E6E6E6

Secondary Text: #9AA0A6

Positive: Green

Negative: Red

Warning / Halt: Amber

Font: Monospace / Condensed (Bloomberg-like density)

Layout Rules

No cards, no rounded corners

Grids, tables, panels

Numbers > charts

Hover = reveal details

Click = opens panel, not modal
